Sequence Manipulation Suite
===========================

The Sequence Manipulation Suite is a collection of JavaScript programs for generating, formatting, and analyzing short DNA and protein sequences. It is commonly used by molecular biologists, for teaching, and for program and algorithm testing. It was built based on `The Sequence Manipulation Suite <https://pubmed.ncbi.nlm.nih.gov/10868275/>`__.
